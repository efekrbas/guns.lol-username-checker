# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **Do NOT open a public issue.**
2. Email the maintainer or use [GitHub's private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability).
3. Include a detailed description of the vulnerability and steps to reproduce it.
4. You will receive a response within **48 hours** acknowledging the report.

## Scope

This project is a CLI tool that makes HTTP requests to `guns.lol`. It does **not** collect, store, or transmit any personal user data. Webhook URLs provided by users are only used locally to send Discord notifications and are never logged or shared.

Thank you for helping keep this project safe!
